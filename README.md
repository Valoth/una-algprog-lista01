# Algoritmos Narrativos: Lógica e Tomada de Decisão

Este repositório reúne resoluções de desafios de lógica estruturados em algoritmos narrativos. O foco desta atividade foi decompor problemas complexos em passos sequenciais e aplicar estruturas condicionais (Se/Então/Senão) para resolver situações do cotidiano e dilemas clássicos de lógica.

## Desafios Propostos

### 1. O Enigma da Travessia
Resolução do clássico problema de transporte de carga em um barco com capacidade limitada. O objetivo foi cruzar um rio com um lobo, uma cabra e um maço de couve, respeitando as restrições de convivência entre os itens sem supervisão.

### 2. Fluxo de Compra: E-commerce
Desenvolvimento do passo a passo para a aquisição de um smartphone em uma plataforma online. O algoritmo cobre desde a validação de segurança do vendedor até a verificação de limites financeiros e escolha de frete.

### 3. Navegação Urbana: Estação Novo Eldorado
Um algoritmo de deslocamento inteligente que avalia variáveis externas como condições climáticas e distância geográfica para decidir o melhor meio de transporte até a estação de metrô.

## Competências Exercitadas
* **Decomposição:** Divisão de problemas em etapas menores.
* **Pensamento Condicional:** Aplicação de lógica para diferentes cenários.
* **UX de Processo:** Criação de fluxos que fazem sentido no mundo real.
* **Documentação:** Organização de portfólio técnico no GitHub.

## Como ler este repositório
Os algoritmos estão detalhados no arquivo de resolução, utilizando uma linguagem clara e humanizada, focando na coerência lógica necessária para o desenvolvimento de software e automação de processos.
